# Matriz de Avaliacao e Evidencias

## Bloco 1: Gerenciamento do projeto

Evidencias:

- quadro Kanban no Agiflow;
- colunas descritas;
- cards com descricao, prioridade, criterios de conclusao e estado;
- explicacao do processo iterativo e incremental.

Artefatos relacionados:

- `docs/01-gerenciamento-kanban.md`
- Agiflow: tarefas `WXX-1` a `WXX-3`

## Bloco 2: Vision

Evidencias:

- problema;
- posicionamento do sistema;
- stakeholders;
- ambiente de uso;
- necessidades;
- funcionalidades resumidas;
- requisitos nao funcionais resumidos;
- solucao proposta.

Artefatos relacionados:

- `docs/02-vision-scope.md`
- Agiflow: `WXX-4`, `WXX-5`

## Bloco 3: Requisitos nao funcionais

Evidencias:

- requisitos funcionais complementares que nao estejam descritos nas historias;
- usabilidade;
- desempenho;
- seguranca;
- requisitos de interface com usuario;
- interface com dispositivos externos;
- interface com outros sistemas;
- interfaces externas;
- documentacao;
- licenciamento;
- conformidade;
- restricoes de projeto.

Artefatos relacionados:

- `docs/03-requisitos-nao-funcionais.md`
- Agiflow: `WXX-6`

## Bloco 4: Historias de usuario

Evidencias:

- historias no formato "Como..., eu quero..., para...";
- linguagem centrada no usuario;
- cobertura dos servicos obrigatorios.

Artefatos relacionados:

- `docs/04-historias-de-usuario.md`
- Agiflow: `WXX-7`, `WXX-8`

## Bloco 5: Arquitetura

Evidencias:

- objetivos arquiteturais;
- decisoes;
- justificativas;
- restricoes;
- dependencias;
- mecanismos e abstracoes.

Artefatos relacionados:

- `docs/05-architecture-notebook.md`
- Agiflow: `WXX-9`, `WXX-10`

## Bloco 6: UX

Evidencias:

- storyboards dos cenarios principais;
- cada storyboard composto por sequencia de wireframes;
- cada wireframe como esboco simples de tela;
- revisao de coerencia com requisitos e arquitetura.

Artefatos relacionados:

- `docs/07-ux-storyboards-wireframes.md`
- `ux/`
- Agiflow: `WXX-13` a `WXX-15`

## Bloco 7: Banco de dados

Evidencias:

- diagrama fisico;
- tabelas;
- colunas;
- tipos;
- chaves primarias e estrangeiras;
- relacionamentos;
- proposito textual de cada tabela.

Artefatos relacionados:

- `docs/06-projeto-fisico-banco.md`
- Agiflow: `WXX-11`, `WXX-12`

## Bloco 8: Prototipo funcional

Evidencias:

- prototipo de acordo com o projeto de interface e arquitetura;
- CRUD completo;
- movimentacao entre colunas;
- movimentacao entre raias;
- movimentacao por arrastar e soltar;
- limite WIP;
- metricas Kanban;
- integracao entre interface, regra de negocio e persistencia.

Artefatos relacionados:

- `prototype/`
- Agiflow: `WXX-16` a `WXX-25`, `WXX-33` a `WXX-35`
